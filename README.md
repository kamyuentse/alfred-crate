Alfred workflow to search crates in [crates.io][crates-io]

## Usage

| Key  | Action                                          |
| :-:  | :-----                                          |
| `↩`  | Open the crate's page on [crates.io][crates-io] |
| `⌘↩` | Send config entry to clipboard                  |
| `⌥↩` | Open documentation of the crate                 |

---

[crates-io]: https://crates.io
